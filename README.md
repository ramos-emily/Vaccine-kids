# VacinaKids

Plataforma digital para acompanhamento da vacinação infantil, substituindo a carteira física de vacinação e facilitando o controle de imunização de crianças.

---

## Demonstração

Acesse o projeto online:
https://vaccine-kids.vercel.app/home

---

## Sobre o projeto

O VacinaKids é uma aplicação desenvolvida com **Ionic + Angular (Standalone)** que permite:

- Cadastro e acompanhamento de crianças
- Visualização do histórico vacinal
- Controle de vacinas aplicadas, pendentes e atrasadas
- Acompanhamento de campanhas de vacinação
- Dashboard com situação vacinal

---

## Funcionalidades

###Crianças
- Lista de crianças cadastradas
- Detalhes individuais
- Histórico de vacinas por criança

### Vacinas
- Catálogo completo de vacinas
- Informações por idade e doses
- Status de aplicação

### Campanhas
- Campanhas ativas de vacinação
- Informações de público-alvo
- Período de aplicação

### Dashboard
- Percentual de vacinação concluída
- Vacinas pendentes
- Vacinas atrasadas

---

## Interface

O projeto utiliza uma identidade visual baseada na paleta:

- Verde: `#ABC270`
- Amarelo: `#FEC868`
- Laranja: `#FDA769`
- Marrom escuro: `#473C33`

Interface responsiva para desktop, tablet e mobile.
